📚 PROCHAINES ÉTAPES
═══════════════════════════════════════════════════════════════════════════════

PHASE 1 (LOCAL) — 30 minutes:
  1. Éditer .env avec tes clés API
  2. python tests/test_local.py (chat terminal)
  3. Tester différents scénarios

PHASE 2 (SERVEUR) — 10 minutes:
  1. uvicorn agent.main:app --reload --port 8000
  2. curl http://localhost:8000/health
  3. Vérifier que tout fonctionne

PHASE 3 (WHATSAPP) — Configurer le webhook:
  Meta: https://developers.facebook.com → WhatsApp → Configuration
  Twilio: https://console.twilio.com → WhatsApp Sandbox

PHASE 4 (PRODUCTION) — Déployer sur Railway:
  1. git push vers GitHub
  2. Connecter Railway (déploiement auto)
  3. Configurer les variables d'environnement
